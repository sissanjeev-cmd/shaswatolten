LLM × Robotics · Orangewood Labs · 2023
RoboGPT
An LLM-powered robotics programming tool that reduced new use-case programming time by 50%. Added audio and textual awareness using LangChain and ChatGPT APIs. Integrated zero-shot object detection and YOLO for visual understanding. Plugged into MoveIt + Gazebo simulation pipeline, cutting deployment time from 1 week to just 2 days.

Integrated environment pointcloud as Octomap (via RTAB-Map) in MoveIt for robot-world collision checking during motion planning.
LangChain ChatGPT API YOLO Zero-Shot Detection MoveIt Gazebo ROS2 RTAB-Map Octomap

MASc Thesis · UWaterloo · 2024–25
UAV Path Planning via Safe Reinforcement Learning
Vision-based path planning and control of UAVs and Aerial Manipulators using Safe RL. Benchmarked off-policy RL algorithms for path planning using LiDAR — accepted at ICUAS 2024. Developing RL trajectory tracking using 3D Point Cloud in complex obstacle environments.
Safe RL LiDAR 3D Point Cloud ICUAS 2024 Python
🦾
Safe RL · TMU MITACS · 2022
Safe RL for Continuum Robot Control
Implemented DDPG, TD3, SoftQ, SAC, NAF and constraint-based Safety Layer from scratch using PyTorch. Proposed a constraint-based Safe RL method using SAC + CMA to promote efficient exploration while ensuring robot safety. Built custom Gym environment based on Cosserat-rod/string models.
PyTorch SAC / TD3 CMA OpenAI Gym Cosserat Model

Musculoskeletal RL · UToronto · 2024
RL for Bionic Legs Control
Developed reinforcement learning algorithms for control of musculoskeletal systems with Bionic Legs at the University of Toronto Robotics Institute, Neural Robotics Lab. Bridging RL and biomechanics.
RL Bionic Legs Musculoskeletal PyTorch

Simulation Tool · IIT Delhi · 2021–22
Railhunt — Railway Stability Simulator
User-friendly simulation tool built with MATLAB App-Designer to study hunting stability of railway vehicles. Studied effects of dissimilar rail properties using the DeNOC method for a 17-DOF rail model.
MATLAB App-Designer DeNOC 17-DOF Model

Developed an ANN model achieving 99.89% accuracy for predicting end milling parameters for Al6061 alloy. Presented at the International Conference of Advanced Manufacturing and Materials Processing (ICAMMP 2021).
